```
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract TokenBridge {
    event TokenMinted(address indexed token, address indexed recipient, uint256 amount);

    function lockToken(address token, uint256 amount) public {
        // Lock the token on this chain
        require(token.transfer(amount), "Token transfer failed");

        // Emit an event for minting the token on the other chain
        emit TokenMinted(token, msg.sender, amount);
    }
}
```
This contract has a single function, `lockToken`, which locks a specified amount of tokens on the current chain and emits an event for minting the same amount of tokens on the other chain. The event is emitted with the token address, the recipient address, and the amount of tokens.

Note that the `transfer` function is used to lock the tokens on the current chain, and the `emit` statement is used to emit the event. The `msg.sender` variable is used to get the address of the sender of the transaction, which is the recipient of the minted tokens on the other chain.
